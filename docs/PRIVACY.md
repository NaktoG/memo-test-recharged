# Privacy

## Datos Locales

La aplicacion no usa backend ni envia datos a terceros.

Se guardan en `localStorage` del navegador:

- nickname elegido por el jugador,
- idioma preferido,
- preferencia de sonido,
- estadisticas locales de partidas.

## Clasificacion

- Nickname: dato de personalizacion local. Puede ser personal si el usuario decide escribir informacion identificable.
- Estadisticas: dato operativo local.
- Preferencias: dato tecnico local.

## Minimizacion

No se solicita email, nombre real, cuenta, token ni informacion sensible.

## Eliminacion

El usuario puede cambiar jugador desde la pantalla inicial y resetear estadisticas desde el panel de progreso.
